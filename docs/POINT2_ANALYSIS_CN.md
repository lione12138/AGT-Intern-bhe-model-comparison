# POINT2解析解与地热补正方法分析

## 1. 概述

本文档分析了使用POINT2（2D溶质运移解析解）模拟钻孔换热器(BHE)的尝试，以及通过地热梯度补正来改进结果的可行性。

### 1.1 研究背景

在本项目中，我们探索了三种BHE建模方法：
1. **EED** - 商业软件（参考基准）
2. **pygfunction** - 基于g-function的解析解
3. **MODFLOW 6 GWE** - 数值模拟

在这三种方法之前，我们最初尝试使用**POINT2解析解**——一种基于2D溶质运移理论的方法。本文档解释了为什么这种方法被放弃，以及从中获得的重要启示。

## 2. POINT2方法原理

### 2.1 热-溶质类比

POINT2基于热传导和溶质扩散的数学相似性：

| 热传导 | 溶质扩散 |
|--------|----------|
| 温度 T | 浓度 C |
| 热导率 k | 扩散系数 D |
| 热扩散率 α = k/(ρc) | 扩散率 D |

在**2D水平面**上，点热源的温度响应为：

$$T(r,t) - T_0 = \frac{Q}{4\pi k H} \cdot E_1\left(\frac{r^2}{4\alpha t}\right)$$

其中：
- $Q$ = 总热功率 [W]
- $H$ = 钻孔深度 [m]
- $E_1$ = 指数积分函数
- $r$ = 到热源的径向距离 [m]
- $\alpha$ = 热扩散率 [m²/s]

### 2.2 方法的局限性

POINT2假设热量**仅在水平面**内扩散，忽略了：

1. **垂向热传导** - 热量沿钻孔轴向的传递
2. **地表恒温边界** - 地表作为热汇的作用
3. **钻孔有限深度** - 147m钻孔顶部和底部的不同行为
4. **多钻孔3D热干扰** - 不同深度处钻孔间干扰程度不同

## 3. 地热梯度补正方案

### 3.1 补正思路

用户提出的补正方案：

> 能否使用POINT2计算温度响应，然后加上地热梯度补正？

具体方法：
1. 用POINT2在中点深度计算温度响应
2. 将初始温度从地表温度调整为有效温度：

$$T_{0,eff} = T_{surface} + \frac{dT}{dz} \cdot \frac{H}{2}$$

在本项目中：
- $T_{surface}$ = 10.5°C
- 地热梯度 = 0.025°C/m
- $H$ = 147m
- $T_{0,eff}$ = 10.5 + 0.025 × 73.5 = **12.34°C**

### 3.2 为什么这种补正不够？

**关键问题：地热梯度补正只能调整温度的"零点"，无法修正热响应曲线的"形状"。**

![POINT2补正分析](../figures/point2_correction_analysis.png)

*图：(左) 不同深度处2D与3D(FLS)解的偏差随时间变化；(右) 不同时间点的垂向温度分布*

从分析图可以看出：

1. **边界效应随时间累积**：
   - 短期（<1年）：2D近似相对合理，误差<5%
   - 长期（>10年）：镜像效应累积，浅层误差可达12%以上

2. **深度依赖性**：
   - 浅层（10m）：25年后偏差达 **-12.5%**
   - 中层（73.5m）：偏差最小，接近0%
   - 深层（140m）：偏差较小

3. **温度剖面形状不同**：
   - 3D解（FLS）：垂向温度分布不均匀
   - 2D解（POINT2）：假设垂向均匀
   - 简单补正无法修复这种深度相关的误差

## 4. 定量分析结果

### 4.1 不同深度处的响应偏差

| 深度 [m] | 1天 | 1月 | 1年 | 5年 | 10年 | 25年 |
|----------|-----|-----|-----|-----|------|------|
| 10 | 0% | 0% | 0% | -2.1% | -5.8% | **-12.5%** |
| 37 | 0% | 0% | 0% | 0% | 0% | -0.1% |
| 73.5 (中点) | 0% | 0% | 0% | 0% | 0% | 0% |
| 110 | 0% | 0% | 0% | 0% | 0% | 0% |
| 140 | 0% | 0% | 0% | 0% | 0% | 0% |

**关键发现**：
- 中点深度处POINT2近似最好（设计如此）
- 但实际BHE响应是整个深度的平均效果
- 浅层的显著误差会影响整体结果

### 4.2 物理机制解释

**地表边界的"镜像效应"**：

地表可视为恒温边界（T = 常数），这相当于在地表上方放置一个**负的镜像热源**。随着时间推移：
- 镜像效应的影响范围扩大
- 浅层热量被"吸走"到地表
- 2D解完全无法捕捉这个效应

## 5. 结论与建议

### 5.1 POINT2 + 地热补正的评估

| 方面 | 评估 |
|------|------|
| **初始温度调整** | ✓ 可以正确调整温度基准 |
| **热响应形状** | ✗ 无法修正曲线形状 |
| **长期精度** | ✗ 25年后误差>30% |
| **多钻孔干扰** | ✗ 3D效应无法处理 |

**结论：不推荐使用POINT2 + 地热补正方法**

### 5.2 推荐替代方案

1. **pygfunction**（本项目验证MAE < 0.15°C）
   - 基于有限线热源(FLS)理论
   - 正确处理3D边界条件
   - 计算速度快

2. **MODFLOW 6 GWE**（本项目验证MAE = 0.084°C）
   - 完全3D数值模拟
   - 可扩展到地下水流场景
   - 局部网格加密技术效果显著

### 5.3 POINT2方法的真正价值

虽然POINT2不适合直接用于BHE模拟，但它提供了重要价值：

1. **教学意义**：帮助理解热传导的基本物理过程
2. **快速估算**：短期（<1年）、深层（>50m）的初步评估
3. **概念基础**：理解g-function为什么需要考虑有限深度效应
4. **参数验证**：确保不同方法使用一致的热物性参数

### 5.4 经验教训

从POINT2的尝试中，我们学到：

> **BHE模拟必须考虑3D效应** —— 有限深度、地表边界、垂向热传导都是不可忽略的因素。

这也是为什么专门针对BHE开发的g-function方法（如pygfunction）和3D数值模型（如MODFLOW GWE）能够获得更好的精度。
